这个项目是需要完成求职的完整工作流。
这项目需要有几个skill，供ai agent工作，还要有agent的prompt，最后要生成一个完整求职工作流的agent。
首先第一个skill要根据我的项目经验和个人信息写一个通用简历，需要符合中国企业的喜好编写。需要使用流程的中文，显示出专业性，不得夸大其词，过于自大。必须谦逊有礼，符合求职要求。skillhub上有大量相关skill，请参考！
然后第二个skill要对中国科技大厂进行岗位搜索，先进行初步搜索，需要根据简历进行相关的岗位搜索，搜索有什么公司和相关岗位，然后对用户进行询问喜好的公司、城市和岗位。最后整理成一个md文档,详细列出公司名、城市、岗位和岗位信息和要求。
然后第三个skill是根据相关岗位，进行细致的简历微调，并生成一个面试问题，根据该岗位和简历的信息，其中面试问题需要在牛客网和小红书上进行搜索，这里需要下载牛客和小红书的siill，skillhub上有。
最后是投递skill，使用模拟浏览器浏览该公司的投递官网，把需要填写的信息逐项写到md里，供用户一键复制填写。
